# Lesson Takeaways

You learned that:

⭐ `@for` is used to create a list of items

⭐ The `@empty` block can be used when the list is empty

⭐ `@if` is used to add or remove elements based on the condition

⭐ `@if` can have `@else if` and `@else` blocks
